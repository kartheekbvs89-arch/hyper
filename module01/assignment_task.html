Assignment Overview

In this assignment you will empirically reproduce every key claim from Module 01 by sweeping two hyperparameters of two simple models on the load_digits dataset, identifying the underfitting/sweet-spot/overfitting zones on the resulting curves, and computing a numerical bias-variance decomposition using bootstrap resampling. The bonus task introduces TimeSeriesSplit and asks you to explain why the resulting curves differ from the standard k-fold versions.

The assignment is designed to take 2–3 hours if you are comfortable with numpy, pandas, and matplotlib. If you find yourself spending more than 4 hours, re-read the part 2 and part 3 articles — most stuck points come from a shaky grasp of the bias-variance math or the CV loop structure, not from the code itself.

Dataset

We use sklearn.datasets.load_digits — the classic 8×8 handwritten digits dataset, 1,797 samples, 64 features (pixel intensities 0–16), 10 classes (digits 0–9). It is small enough to run every experiment in seconds, large enough that CV estimates are reasonably stable, and rich enough that the bias-variance U-curve is clearly visible.

Task 4 (Bonus) — TimeSeriesSplit on a Randomly-Shuffled Dataset

Repeat Task 1 but use TimeSeriesSplit(n_splits=5) instead of StratifiedKFold. The digits dataset is not temporal, so the difference won't be a leakage effect — instead, you should observe that TimeSeriesSplit's folds have unequal training set sizes (the first fold trains on only 1/6 of the data; the last fold trains on 5/6), which systematically biases early folds toward underfitting.
