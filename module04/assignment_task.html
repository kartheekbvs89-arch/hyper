Assignment Overview

In this assignment you will build, tune, and analyze an end-to-end XGBoost model on the UCI Adult Census dataset — the same dataset we used as a worked example in Part 1. The goal is to put into practice the full gradient boosting tuning workflow you learned in Part 3: pipeline construction, the triad (learning_rate × n_estimators × max_depth) with early stopping, the regularization knobs with RandomizedSearchCV, and feature importance analysis. The bonus task introduces LightGBM and asks you to compare it head-to-head with XGBoost under matched hyperparameters.

Estimated time: 3–4 hours if you are comfortable with scikit-learn pipelines and have XGBoost installed. If you find yourself spending more than 5 hours, re-read Part 3 sections 3, 4, and 11 — most stuck points come from skipping the early-stopping step or from a malformed pipeline.

You will need the following packages installed: numpy, pandas, scikit-learn (1.4+), xgboost (2.0+), lightgbm (4.0+), matplotlib. Install with pip install numpy pandas scikit-learn xgboost lightgbm matplotlib.

Dataset: UCI Adult Census

We use the UCI Adult Census dataset, fetched via scikit-learn's fetch_openml(name='adult', version=2). It has 48,842 rows and 14 mixed-type features (5 numeric, 9 categorical). The target is binary: <=50K (76% of samples) vs >50K (24%). This is a realistic imbalanced binary classification problem with enough samples that hyperparameter choices matter, but small enough that each XGBoost fit takes only a few seconds.

In [1]: python
>>> # Load Adult Census and basic EDA
from sklearn.datasets import fetch_openml
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

adult = fetch_openml(name='adult', version=2, as_frame=True, parser='auto')
X = adult.data
y = adult.target

print(f"X.shape = {X.shape}")
print(f"target distribution:\n{y.value_counts(normalize=True).round(3)}")
print(f"\nfeature dtypes:\n{X.dtypes}")
print(f"\nmissing values per column:\n{X.isna().sum()[X.isna().sum() > 0]}")

# Drop rows with missing values for simplicity (in production, impute instead)
mask = ~X.isna().any(axis=1)
X, y = X[mask], y[mask]
print(f"\nAfter dropping missing: X.shape = {X.shape}")

# Encode target as 0/1
y = (y == '>50K').astype(int)
