Assignment Overview

This assignment brings together every concept from Module 05. You will build a PyTorch CNN for CIFAR-10, write an Optuna objective that searches over learning rate, batch size, dropout, number of filters, weight decay, and optimizer choice, run a 50-trial study with the MedianPruner, visualize the results four different ways, retrain the best configuration for 30 epochs with cosine annealing, and (bonus) implement linear warmup + cosine decay and compare against plain cosine.

By the end you will have built a small but complete deep-learning tuning pipeline — the same skeleton you will reuse on real projects. Expect to spend 4–5 hours total: about 30 minutes on each of the six tasks, plus debugging time.

📌
Hardware requirements

A GPU is strongly recommended but not required. Each 5-epoch trial takes ~3 minutes on a single modern GPU (RTX 3060 or better) and ~10–15 minutes on CPU. The full 50-trial study with MedianPruner typically takes 1–2 hours on GPU. If you only have CPU, reduce n_trials to 20 and epochs_per_trial to 3.

Dataset — CIFAR-10

CIFAR-10 is 60,000 32×32 RGB images in 10 classes (airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck). 50,000 are training, 10,000 are test. It is small enough to train in minutes per trial but hard enough that architecture and regularization choices matter — a badly-tuned CNN gets ~60% test accuracy, a well-tuned one gets 90%+.

Use torchvision.datasets.CIFAR10 with the standard normalization. Hold out 5,000 images from the training set as validation for the Optuna objective; keep the test set untouched until the final retraining in Task 5.

cifar10_setup.py
# ─── cifar10_setup.py ──────────────────────────────────────────────────────────
import torch
from torch.utils.data import DataLoader, random_split
from torchvision import datasets, transforms

# Standard CIFAR-10 normalization (per-channel mean/std).
tfm_train = transforms.Compose([
    transforms.RandomHorizontalFlip(),
    transforms.RandomCrop(32, padding=4),
    transforms.ToTensor(),
    transforms.Normalize((0.4914, 0.4822, 0.4465),
                         (0.2470, 0.2435, 0.2616)),
])
tfm_eval = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.4914, 0.4822, 0.4465),
                         (0.2470, 0.2435, 0.2616)),
])

full_train = datasets.CIFAR10("./data", train=True, download=True, transform=tfm_train)
test_set   = datasets.CIFAR10("./data", train=False, download=True, transform=tfm_eval)

# 45k train / 5k val split (fixed seed for reproducibility).
train_ds, val_ds = random_split(
    full_train, [45000, 5000],
    generator=torch.Generator().manual_seed(42),
)

print(f"train: {len(train_ds)}, val: {len(val_ds)}, test: {len(test_set)}")

  • learning_rate — log-uniform in [1e-5, 1e-1]
  • batch_size — categorical [32, 64, 128, 256]
  • dropout — uniform [0.0, 0.5]
  • n_filters — int [32, 128] step 16
  • weight_decay — log-uniform [1e-6, 1e-2]

Task 3 — Run 50 Trials with MedianPruner

Create an optuna.create_study(direction="maximize", pruner=optuna.pruners.MedianPruner(n_startup_trials=5, n_warmup_steps=1)) and run study.optimize(objective, n_trials=50). Time the run. Print study.best_value and study.best_params at the end.

After the study completes, also report len([t for t in study.trials if t.state == optuna.trial.TrialState.COMPLETE]) and len([t for t in study.trials if t.state == optuna.trial.TrialState.PRUNED]) to see how aggressive the pruner was.

In [3]: python
>>> # YOUR CODE HERE
import time
import optuna

study = optuna.create_study(
    direction="maximize",
    pruner=optuna.pruners.MedianPruner(n_startup_trials=5, n_warmup_steps=1),
)

t0 = time.time()
study.optimize(objective, n_trials=50)
elapsed = time.time() - t0

print(f"Best val acc: {study.best_value:.4f}")
print(f"Best params: {study.best_params}")
print(f"Elapsed: {elapsed/60:.1f} min")

# Count trial states
from collections import Counter
states = Counter(t.state for t in study.trials)
print(f"Trial states: {dict(states)}")

Expected output: Best val accuracy around 0.80–0.85 (5 epochs is short for CIFAR-10). Roughly 30–40 of the 50 trials should complete; the rest pruned. Total runtime ~1 hour on GPU.
