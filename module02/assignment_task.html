Assignment Overview

In this assignment you will design, run, and profile a coarse-to-fine grid search on a RandomForestClassifier trained on the UCI Adult Census dataset. The goal is to internalize the practical workflow: pick sensible ranges, run a coarse search, identify the best region, run a fine search, and produce a publication-quality heatmap plus a stability analysis of the top configurations.

This assignment takes 2–3 hours. The Adult dataset is moderately large (49k rows) so each fit takes a few seconds; the total search will finish in under 30 minutes.

Dataset

The UCI Adult Census dataset predicts whether a person earns more than $50K/year based on demographic features. We load it from scikit-learn's fetch_openml for convenience.
